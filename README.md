# DecodeLabs Project 3 - AI Recommendation Logic

## 📌 Project Overview
This project is part of the DecodeLabs Artificial Intelligence Internship.  
It is a simple AI-based recommendation system that suggests items based on user interests using similarity and pattern matching logic.

---

## 📁 File Information
- File Name: project3.py
- File Type: Python (.py)

---

## 🚀 How to Run
Run the program using:
python project3.py

---

## ⚙️ Features
- Takes user interests as input
- Matches interests with item tags
- Calculates similarity score
- Sorts recommendations by relevance
- Displays personalized results

---

## 🧠 Concept Used
- AI Recommendation Logic
- Pattern Matching
- Similarity Scoring
- Data Filtering
- Basic Algorithm Design
